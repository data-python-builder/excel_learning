# What-If Analysis

## Overview

This section introduces Excel’s **What-If Analysis tools**, which allow users to test different scenarios and evaluate how changes in inputs affect results. These tools are commonly used in financial modeling, forecasting, and decision-making.

---

## Skills & Concepts Covered

### Data Table Analysis

* Understanding the purpose of **Data Tables** in scenario analysis
* Creating one-variable and two-variable data tables
* Analyzing how input changes impact calculated results

### Goal Seek

* Using **Goal Seek** to find the required input for a desired output
* Solving reverse calculations in financial and business scenarios
* Understanding how Excel iteratively adjusts values to reach a target

### Goal Seek with Formulas

* Applying Goal Seek with formula-based models
* Solving financial and analytical problems by adjusting variables
* Integrating Goal Seek into practical Excel models

---

## Practical Use Cases

* Testing different financial scenarios
* Determining required sales to reach a profit target
* Calculating required interest rates or payments in loan models
* Evaluating the impact of changing variables in business models
* Performing quick scenario simulations for decision making

---

## Notes

This section focuses on developing practical skills in **scenario analysis and decision support using Excel**. The emphasis is on understanding how small changes in inputs affect outputs and using Excel tools to explore different business or financial outcomes efficiently.
