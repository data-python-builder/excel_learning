# Excel Core Fundamentals

## Overview
This section focuses on building strong Excel fundamentals required for day-to-day business, operations, and analytical work. 
The emphasis is on writing reliable formulas, applying logical functions, structuring data cleanly, and creating usable visual outputs.

---

## Skills & Concepts Covered

### Formulas & Cell Referencing
- Relative, absolute, and mixed cell references
- Writing formulas using correct operator precedence (BODMAS)
- Copying and scaling formulas without errors

### Functions & Logical Thinking
- Core functions such as SUM, AVERAGE, and COUNT
- Conditional logic using IF, AND, and OR
- Multi-criteria functions including SUMIFS, AVERAGEIFS, and COUNTIFS
- Text manipulation using CONCATENATE
- Data retrieval using VLOOKUP

### Data Structuring & Visualization
- Formatting datasets using Excel Tables
- Creating and formatting basic charts
- Organizing data for clarity and scalability

### Workbook Navigation
- Building internal navigation between worksheets
- Improving usability for large workbooks

---

## Practical Use Cases
- Sales and performance tracking
- Operations and inventory reporting
- MIS and summary dashboards
- Structured datasets for analysis

---

## Notes
The focus of this section is on developing logical thinking and clean structure in Excel, ensuring spreadsheets remain accurate, scalable, and easy to use.
